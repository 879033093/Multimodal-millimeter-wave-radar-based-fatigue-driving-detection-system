The open-source data includes:  
1. Eye movement data  
2. Example code for extracting raw eye movement data  
3. Head fatigue movement micro-Doppler images  
4. Example code for extracting micro-Doppler images  
5. Respiratory and heartbeat differential data  
6. Example code for extracting raw respiratory and heartbeat data  
7. Fatigue driving detection results

Data analysis tool: MATLAB  
The complete code and analysis will be continuously updated.
